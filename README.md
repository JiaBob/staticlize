# Staticlize
Make your website build from dynamic web framework static

require Python 3.6 with urllib3 

### Run

Open your server on your local machine, make sure your website is on 127.0.0.1

Run ```python getStatics.py``` then you can find your website is localized near the place of the script.

Note: It only localize your .html file. Please move your static files manually.
